# Osgood OS v1.0.0 — Operations Foundation

This release establishes Osgood OS as a unified venue operations platform.

## Main release features

- A dedicated Operations page for building and internal responsibilities.
- One accountable owner for every task.
- Mandatory due dates and automatic overdue visibility.
- Priority and progress tracking.
- Blocked-task accountability.
- Dashboard visibility for open, due-today, overdue, and upcoming building work.
- Existing venue sales, event, proposal, payment, and floor-planning tools preserved.

## Required database step

Run `supabase/migrations/008_operations_center_tasks.sql` in the Supabase SQL Editor before testing task creation.
